## 项目使用说明

安装好需要的库文件，具体的在requirement.txt中。

1. bird.py：FlappyBird的训练代码。
2. mountain_car.py：MountainCar-v0的训练代码
3. car_model.h5：已经训练好的MountainCar-v0模型 
4. test_mountain_car.py：测试MountainCar-v0的代码
